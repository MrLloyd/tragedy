# 惨剧轮回 — Claude 工作指南

## 推荐技能：代码定位与 Token 节省

### Skill 1: 精确定位代码片段

可选的技巧组合（推荐但不强制）：

```
(1) rg "pattern" -n          # 定位行号
(2) Read offset=N limit=50   # 精确读取片段
(3) 或 Grep "pattern"        # 直接搜索，不读整个文件
```

### Skill 2: 任务隔离保护上下文

探索型任务可以交给 Explore subagent：
- 代码结构理解
- 跨文件分析
- 模式查找

这样主上下文保持干净，token 开销分散。

### Skill 3: 分块读取

- 单次 Read 在 50 行以内较高效
- 如果需要看全貌再读全文
